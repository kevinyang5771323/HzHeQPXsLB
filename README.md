# 企业资产管理系统 java247

## 项目概述

企业资产管理系统是一款用于企业内部资产管理的软件，通过高效的信息化手段，实现资产的分类、信息管理、借还流程以及维修跟踪等功能。

## 技术栈

### 开发工具

- IntelliJ IDEA 2020.3
- Visual Studio Code

### 运行环境

- JDK 1.8
- Maven 3.6.0
- MySQL 5.7
- Node.js 14

### 服务端技术

- Spring Boot
- MyBatis-Plus

### 前端技术

- HTML
- CSS
- Layui
- jQuery
- Bootstrap
- Vue.js
- Axios
- Element-UI

## 功能模块

### 系统角色

- 管理员
- 用户

#### 管理员角色功能

- 个人中心管理
- 用户管理
- 资产分类管理
- 资产信息管理
- 资产借出管理
- 资产归还管理
- 资产维修管理

#### 用户角色功能

- 资产借出
- 资产归还

## 系统部署

### 目录结构

```
|-- 项目根目录
    |-- src
        |-- main
            |-- java
                |-- com.example
                    |-- controller
                    |-- service
                    |-- mapper
                    |-- entity
            |-- resources
                |-- application.properties
            |-- webapp
                |-- static
                |-- WEB-INF
                    |-- web.xml
    |-- pom.xml
    |-- package.json
    |-- ...
```

### 部署步骤

1. 准备上述运行环境
2. 克隆或下载项目源码
3. 使用 Maven 构建项目，并生成可执行的 JAR 文件
4. 部署至支持 Java 的 Web 服务器（如 Tomcat）
5. 运行数据库迁移脚本，创建数据库和表结构
6. 启动应用

## 核心亮点

- 采用前后端不分离的架构，易于维护与部署
- 基于成熟的 Spring Boot 和 Vue.js 技术栈，系统稳定可靠
- 完善的权限管理，区分管理员和用户角色，实现不同功能模块的访问控制
- 界面友好，操作简便，提高企业资产管理效率
